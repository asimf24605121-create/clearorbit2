AI DETECTION LAYER (IMPORTANT)

1. Field Classification Engine

After parsing key-value pairs, classify each field into categories:

* identity → name, email, phone
* account → plan, country, memberSince
* billing → price, payment method, card
* session → NetflixId, SecureNetflixId, tokens
* metadata → source, labels
* unknown → anything else

Store result like:
{
key: "country",
value: "Brazil 🇧🇷",
category: "account"
}

---

2. Smart Priority Detection

Assign importance score:

HIGH:

* cookies (NetflixId, SecureNetflixId)
* tokens
* login URLs

MEDIUM:

* plan, country, email

LOW:

* labels, source, icons

Sort output by priority automatically

---

3. Context Understanding

Detect patterns:

IF value contains:

* "NetflixId=" → mark as COOKIE
* "nftoken=" → mark as LOGIN_TOKEN
* "@" → mark as EMAIL
* "+" and numbers → PHONE
